## ⚙️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Santhosh7jr/Web_Development.git
   ```

2. Navigate to the directory:

   ```bash
   cd Web_Development
   ```

3. For backend projects:

   ```bash
   npm install
   npm start
   ```

4. Open in browser:

   * Visit `http://localhost:3000`
   * OR open `index.html` directly
